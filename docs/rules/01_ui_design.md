# UIデザインルール

## 基本コンセプト

GanttForgeは、AI駆動開発に適した業務向けガントチャートアプリとする。

以下を最優先とする。

1. 視認性
2. 操作性
3. 一貫性
4. 保守性

---

## デザイン方針

- 白を基調としたシンプルな業務システム
- 無駄な装飾を行わない
- 情報量は多くても見やすいUI
- PC利用を前提とした設計
- 画面サイズが変わっても作業性を維持する
- ページ全体に横スクロールを発生させない
