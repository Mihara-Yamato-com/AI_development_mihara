# GantForge 仕様書

## プロジェクトについて

Webアプリケーション（GantForge）を、AI駆動開発（Claude / Claude Code）で再現することを目的とした学習用プロジェクト。大まかな仕様としてはスケジュール管理アプリ。プロジェクトを追加して、スケジュール項目にやるべき作業と期日を追加。そうすれば横軸のスケジュール表に反映され可視化される。他にはプロジェクト概要の編集やマイルストーンの設定、変更履歴、マインドマップやコメント、即時メモ、タスク管理（未着手・進行中・完了）、コメント、スナップショットなどの追加機能がある。またPDF出力やjson形式で出力、データの取り込みなども行うことができる。

---

## 機能一覧

### ヘッダー
| No | 機能 | 概要 |
|---|---|---|
| H1 | プロジェクト選択 | 登録済みプロジェクトをドロップダウンで切替 |
| H2 | 新規プロジェクト作成 | プロジェクトを追加 |
| H3 | プロジェクト削除 | 選択中のプロジェクトを削除 |
| H4 | Excel出力 | スケジュール表をExcel形式で出力 |
| H5 | PDF出力 | 画面をPDF形式で出力 |
| H6 | データ出力(JSON) | プロジェクトデータをJSONで書き出す |
| H7 | データ取込 | JSONファイルからプロジェクトデータを読み込む |

### 操作バー(表示・編集系)
| No | 機能 | 概要 |
|---|---|---|
| T1 | ロック | プロジェクト全体を編集不可にする |
| T2 | スケジュール追加 | 作業内容・期日を新規登録 |
| T3 | 期間表示切替 | 日/週/1ヶ月/3ヶ月でガントチャートの表示単位を切替 |
| T4 | 今日 | ガントチャートを今日の日付までスクロール |
| T5 | 全て開く | 折りたたまれたスケジュール項目を全展開 |
| T6 | 列表示切替(開始/終了/状態) | 左側スケジュール表の列表示をON/OFF |
| T7 | マイルストーン設定 | 特定日付に区切りを設定(モーダル) |
| T8 | プロジェクト概要編集 | プロジェクト名・説明などを編集(モーダル) |

### 右端パネル系
| No | 機能 | 概要 |
|---|---|---|
| P1 | タスク管理 | 未着手/進行中/完了の3カラムで作業を管理(カンバン形式) |
| P2 | マインドマップ | 中央にプロジェクト、そこから子・兄弟ノードを追加 |
| P3 | 変更履歴 | スケジュール変更・ロック操作など全変更のログ |
| P4 | コメント | 各作業にカーソルを合わせて出る編集ボタンから追加・閲覧 |
| P5 | メモ | 題名+詳細をマークダウンで記述、プレビュー画面あり |
| P6 | 即時メモ | マークダウン形式で素早く書ける簡易メモ(出力のみ) |
| P7 | スナップショット | ある時点の状態を保存・一覧・復元 |

### メインエリア
| No | 機能 | 概要 |
|---|---|---|
| M1 | スケジュール表(左) | 作業内容・開始・終了・状態を一覧表示 |
| M2 | ガントチャート(右) | 期日を横棒で可視化、マイルストーンも表示 |

## 画面構成一覧

画面遷移はなく、すべて単一ビュー内の要素として構成:

| No | 要素名 | 表示方式 |
|---|---|---|
| U1 | ヘッダー | 常時表示(最上部固定) |
| U2 | 操作バー | 常時表示(ヘッダー直下固定) |
| U3 | メインエリア(スケジュール表+ガントチャート) | 常時表示 |
| U4 | タスク管理パネル | 右端からスライドイン |
| U5 | マインドマップパネル | 右端からスライドイン |
| U6 | 変更履歴パネル | 右端からスライドイン |
| U7 | コメントパネル | 右端からスライドイン(作業ホバー時の編集ボタン起点) |
| U8 | メモパネル | 右端からスライドイン |
| U9 | 即時メモパネル | 右端からスライドイン |
| U10 | スナップショットパネル | 右端からスライドイン |
| U11 | プロジェクト概要編集 | モーダル(「編集」ボタンから起動) |
| U12 | マイルストーン設定 | モーダル(「マイルストーン」ボタンから起動) |
